# Personal Portfolio Landing Page Template

A simple landing page template for a personal/portfolio website, built with plain HTML and CSS — no JavaScript or frameworks involved.

## 🖼️ Overview

The template includes the following sections:

- **Header** – A top navbar with the site name and navigation links (Services, Projects, Contact), with support for a mobile menu.
- **Landing Section** – A main hero/welcome section.
- **Features** – Three feature cards using Font Awesome icons.
- **Services** – A services section (Graphic Design, Web Design, UI & UX, Web Development) with an illustrative image.
- **Projects** – A gallery of project cards (image + title + description).
- **Contact** – A contact section with an email address and social media icons.
- **Footer** – Page footer.

## 🛠️ Tools & Technologies Used

| Tool / Library | Purpose |
|---|---|
| **HTML5** | Building the page's core structure |
| **CSS3** | Styling and layout (`css/style.css`) |
| **Normalize.css** | Ensuring consistent rendering of HTML elements across browsers |
| **Font Awesome** (`css/all.min.css`) | Icons used in the navbar, services, and social links |
| **Google Fonts** | Roboto and Work Sans fonts |

There's no JavaScript or frontend libraries (like React or Bootstrap) — the design is pure vanilla CSS, likely using Flexbox/Grid.
